# ⌨️ Typing Speed Test

An interactive web application that measures typing speed (WPM) and accuracy.

## 🚀 Features
- ⏱️ Timer-based typing test (30/60/120 sec)
- 📊 Calculates typing speed (WPM)
- 🎯 Accuracy calculation
- 🔄 Random sentence generation
- 📄 Result displayed on separate screen

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Open index.html in browser
2. Select time
3. Click Start and begin typing
